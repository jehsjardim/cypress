# cypress-e2e

### *Pré-requisitos*

    - Node
    - Cypress
    - IDE da sua preferência (no meu caso estou usando o intelliJ IDEA)

### Configurando projeto
    - git clone 
    - npm install --save-dev cypress

### Execução do teste
    - npm run cy:open
            Vai abrir o cypress e assim pode escolher a spec "Nova oferta"
